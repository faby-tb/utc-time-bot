Privacy Policy

Last updated: 2026-06-30

1. Data Collection

This bot does not collect personal data from users.

The only data processed includes:

  • Discord Server IDs
  • Channel IDs (for voice channel clock display)
  • User permissions (to validate command access)

2. Data Storage

Minimal configuration data may be stored locally (e.g., settings.json) to remember which servers have the clock enabled.

3. Data Sharing

No data is shared, sold, or transmitted to third parties.

4. User Control

Server administrators can disable the bot at any time using the provided commands.

5. Contact

For privacy concerns, contact the developer via Discord or GitHub.
